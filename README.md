### Requirement

Python 3.10  
Pip 23  
Poetry 1.4  
PostgreSQL 16.1


### Установка
1. Склонировать
2. В директории проекта выполнить:  
   `make install`  
   `make build`
3. В корень проекта добавить файл .env и прописать DATABASE_URL(строка с параметрами подключения к базе данных) и SECRET_KEY
